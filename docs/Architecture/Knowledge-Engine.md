# Knowledge Engine

Version: 1.0.0

Status: Development

---

# Mục tiêu

Knowledge Engine là trung tâm quản lý tri thức của K86 Pro.

Engine này chịu trách nhiệm lưu trữ, tổ chức, phân loại và cung cấp toàn bộ dữ liệu tri thức để AI Engine, Shopping Assistant Engine và các Engine khác có thể tra cứu, phân tích và sử dụng.

---

# Trách nhiệm

- Quản lý tài liệu.
- Quản lý hướng dẫn.
- Quản lý FAQ.
- Quản lý Prompt.
- Quản lý Workflow.
- Quản lý dữ liệu đào tạo AI.
- Quản lý tài liệu kỹ thuật.
- Quản lý cơ sở tri thức.
- Đồng bộ nguồn dữ liệu.
- Hỗ trợ tìm kiếm tri thức.
