# OpenAPI Engine

Version: 1.0.0

Status: Development

---

# Mục tiêu

OpenAPI Engine là cổng giao tiếp giữa K86 Pro và các dịch vụ bên ngoài.

Engine này chịu trách nhiệm quản lý API, kết nối hệ thống, đồng bộ dữ liệu và đảm bảo việc tích hợp với các nền tảng bên ngoài diễn ra an toàn, ổn định và có khả năng mở rộng.

---

# Trách nhiệm

- Quản lý REST API.
- Quản lý GraphQL API.
- Kết nối AI Provider.
- Kết nối Affiliate API.
- Kết nối WordPress API.
- Đồng bộ dữ liệu.
- Quản lý Webhook.
- Quản lý OAuth.
- Kiểm soát API Rate Limit.
- Quản lý API Version.
