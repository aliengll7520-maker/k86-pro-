# OpenAPI Engine

Version: 1.0.0

Status: Development

# Mục tiêu

OpenAPI Engine chịu trách nhiệm kết nối K86 Pro với các dịch vụ và hệ thống bên ngoài thông qua API.

---

# Chức năng

- Quản lý REST API.
- Kết nối AI Provider.
- Kết nối Affiliate API.
- Kết nối WordPress API.
- Đồng bộ dữ liệu.
- Quản lý Webhook.
