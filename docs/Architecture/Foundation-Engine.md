# Foundation Engine

Version: 1.0.0

Status: Development

---

# Mục tiêu

Foundation Engine là nền móng của toàn bộ K86 Pro.

Engine này chịu trách nhiệm cung cấp các thành phần dùng chung cho toàn bộ hệ thống.

---

# Trách nhiệm

- Khởi tạo Plugin.
- Quản lý Version.
- Quản lý Database.
- Quản lý Settings.
- Quản lý Logger.
- Quản lý Error Handler.
- Security.
- Hooks.
- Loader.
- Assets.

---
