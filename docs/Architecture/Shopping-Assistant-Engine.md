# Shopping Assistant Engine

Version: 1.0.0

Status: Development

---

# Mục tiêu

Shopping Assistant Engine là trung tâm điều phối của K86 Pro.

Engine này chịu trách nhiệm hỗ trợ người dùng tìm kiếm, phân tích, so sánh và lựa chọn sản phẩm phù hợp dựa trên nhu cầu thực tế, đồng thời kết nối dữ liệu từ các Engine khác để đưa ra khuyến nghị minh bạch và khách quan.

---

# Trách nhiệm

- Hiểu nhu cầu người dùng.
- Phân tích yêu cầu mua sắm.
- Đề xuất sản phẩm phù hợp.
- So sánh nhiều sản phẩm.
- Phân tích ưu và nhược điểm.
- Đưa ra khuyến nghị khách quan.
- Điều hướng người dùng đến nền tảng mua hàng phù hợp.
- Điều phối dữ liệu giữa các Engine.
