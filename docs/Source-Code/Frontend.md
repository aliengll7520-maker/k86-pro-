# Frontend

Version: 1.0.0

Status: Development

## Giới thiệu

Frontend chịu trách nhiệm hiển thị mọi nội dung của K86 Pro trên website.

## Trách nhiệm

- Hiển thị Product Box
- Hiển thị Shopping Assistant
- Hiển thị AI Widget
- Hiển thị Affiliate Box
- Hiển thị Shortcode
- Hiển thị Block Gutenberg

## Thành phần

- Templates
- Shortcodes
- Blocks
- Widgets
- CSS
- JavaScript

## Nguyên tắc

- Không xử lý Database trực tiếp.
- Dữ liệu lấy từ Service.
- Tối ưu tốc độ tải.
- Tương thích Theme.
