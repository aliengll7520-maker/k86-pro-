# Folder Structure

Version: 1.0.0

Status: Development

## Giới thiệu

Tài liệu này mô tả cấu trúc thư mục chuẩn của K86 Pro.

Mục tiêu là tổ chức mã nguồn rõ ràng, dễ bảo trì và dễ mở rộng.

## Cấu trúc thư mục

```text
k86-pro/
├── admin/
├── assets/
├── core/
├── database/
├── frontend/
├── helpers/
├── includes/
├── languages/
├── modules/
├── templates/
├── uninstall.php
├── k86-pro.php
└── readme.txt
```

## Nguyên tắc

- Mỗi thư mục đảm nhận một nhiệm vụ riêng.
- Không đặt mã nguồn sai vị trí.
- Module có thể phát triển độc lập.
- Tuân thủ WordPress Coding Standards.

## Mục tiêu

Định nghĩa cấu trúc thư mục chuẩn cho toàn bộ dự án K86 Pro.
