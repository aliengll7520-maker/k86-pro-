# K86 Pro Source Code

Version: 1.0.0

Status: Development

## Giới thiệu

Source Code là tài liệu mô tả toàn bộ cấu trúc mã nguồn của K86 Pro.

Mục tiêu là xây dựng mã nguồn theo kiến trúc module, dễ bảo trì, dễ mở rộng và dễ kiểm thử.

## Nguyên tắc

- Mỗi file chỉ thực hiện một nhiệm vụ.
- Mỗi module hoạt động độc lập.
- Tuân thủ WordPress Coding Standards.
- Không viết mã trùng lặp.
- Ưu tiên khả năng mở rộng.

## Danh sách tài liệu

- Folder Structure
- Bootstrap
- Loader
- Core
- Admin
- Frontend
- API
- Database
- Helpers
- Assets

## Mục tiêu

Thiết kế toàn bộ mã nguồn trước khi bắt đầu lập trình K86 Pro.
